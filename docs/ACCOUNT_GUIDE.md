# Account & Personalization Guide

Radius is built with privacy and ease of use in mind. This guide explains how the "Account" features work.

## Google Sign-In Integration
We use **Google Identity Services** to provide a seamless, passwordless login experience.

### How it works:
1. **Trigger**: Click the "Account" link in the main menu.
2. **Authentication**: Radius uses the standard Google Sign-In prompt. No password is ever stored on our servers.
3. **Personalization**: Once signed in, your profile picture and name will appear in the navigation bar. 
4. **Data Usage**: We only use your basic profile info (name, picture) to personalize the UI.

## Search History
Radius automatically saves your last **5 unique searches** locally on your device.
- **Privacy**: This data never leaves your browser (stored in `localStorage`).
- **Functionality**: Click any "Recent Search" chip below the main search button to instantly reload those parameters.
