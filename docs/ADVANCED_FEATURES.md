# Advanced Features Index

Radius includes several sophisticated features that set it apart from standard flight search engines.

## 1. Price Prediction Engine (Mock)
Every flight result includes a "Buy Now" or "Wait" badge.
- **Logic**: Calculates the price ratio against a historical average. 
- **Indicator**: High-confidence deals are marked with a vibrant green "Buy Now" badge.

## 2. Deal Meter & Value Logic
Visualized as a progress bar on each flight card.
- **Green**: Exceptional value (price is significantly below average).
- **Amber/Slate**: Standard pricing.
- **Trip Est**: includes estimated 3-night hotel data to give a complete picture of travel costs.

## 3. Night Flight (Dark Mode)
A custom-built dark theme that isn't just a color flip:
- **Refined Contrast**: Uses slate-950 and slate-100 for maximum readability.
- **Blur Effects**: Maintains semi-transparent glassmorphism even in dark mode.
- **Auto-Save**: Remembers your theme preference across sessions.

## 4. Reverse Search Algorithm
Filters destinations based on budget first, then categorizes them by continent or worldwide scope.
- **Direct Link**: Seamlessly hands off to Google Flights with pre-filled destination and date data.
