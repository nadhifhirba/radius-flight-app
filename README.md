# Radius — Flight Discovery for the Modern Traveler ✈️

Radius is a "Reverse Search" flight discovery engine designed to help travelers find their next destination based on budget and vibes, rather than just dates and destination names.

![Radius Header](https://images.unsplash.com/photo-1436491865332-7a61a109cc05?q=80&w=1200&auto=format&fit=crop)

## ✨ Features

- **Budget-First Discovery**: Specify your maximum budget and see every possible destination you can fly to.
- **Real-Time Amadeus Pricing**: Integrated with the Amadeus API for live, up-to-date flight costs.
- **Interactive Budget Map**: Explore destinations globally on an interactive map with color-coded price markers.
- **Smart Math Badges**:
  - **Deal Meter**: Instantly see if a flight is a "Great Value" compared to historical averages.
  - **Trip Est**: includes estimated hotel costs to give you the "real" price of a weekend away.
- **Hidden Gem Alerts**: Highlights under-the-radar destinations that offer incredible value.
- **Domestic Toggle**: One-click filtering for travelers wanting to explore their own backyard.

## 🛠 Tech Stack

- **Frontend**: Vanilla JavaScript (ES6+), HTML5, CSS3.
- **UI Architecture**: Tailwind CSS
- **Maps**: Leaflet.js
- **Icons**: Lucide Icons
- **Backend API**: Amadeus Global Distribution System (GDS) via Node.js
- **Deployment**: Netlify Serverless Functions
